PROJECT


OBSTACLES DETECTOR USING ARDUINO NANO

This project is helpful for detecting obstacles.Specially it is used for blind person. When Obstacle is detected by ultrasonic sensor it sends signal to Arduino nano.

Arduino manupulates this data and send information accordingly to output devices like buzzer which can be heard by blind people.As obstacles come closer to the person, the buzz sound of buzzer increases.
